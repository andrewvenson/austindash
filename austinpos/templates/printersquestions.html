Printers

Oh snap we hate printers not printing or worse, printing gibberish. Let's take the proper troubleshooting steps to get this resolved.

My printer won't print

1. Lets make sure the printer is turned on.
  • Trace power cable from the printer to the wall and make sure printer is recieving power.
  • Let's also make sure the printer is turned on by pressing the power button.
  • If there is a green light by power button and no other light proceed to the next steps.
  • If there are error lights. Repeat process. If issue persists send it in for rma, and look for a loaner or printer to buy while it's getting fixed.
2. Confirm all cables are plugged in securely.
  • Trace the white 9 pin serial cable with black pigtail end or usb cable from the back of the printer to the terminal. Pictures below.
  • Make sure cables are plugged in securely by reseating the cable (Unplugging the cable and plugging them back in on both ends)
  • Confirm what comport the cable from the printer is plugged into. (Flip the head of the terminal back like this so you can see the comports.
  • These are what the comports look like. Remember the comport number that is circled in red for later troubleshooting.
  • Try doing a test print. If printer is still not printing proceed to the next
3. Let's navigate to Focus.
  • Note the name of the computer by viewing the name next to station in bottom right hand corner on login screen.
  • Press the wrench, and then press setup in the right bottom hand corner. Displayed below
  • Navigate to locations at the top navigation bar and select.
  • Press the option 'stations' from the dropdown. View diagram below
  • Select your station on the right hand side under stations.
  • Make sure the comport number that we noted earlier is corresponding to each of these numbers highlighted in red.
  • If so move cable to different comport, note the number, and then change comport number to correspond with cable that's plugged in.
  • Let's do a test print.
  • If everything printed you are good to go, if not please call in to support for further troubleshooting.

Printer is printing gibberish

1. Let's Start by turning off the printer.
  • While printer is turned of press and hold the feed button.
  • While holding the feed button turn the printer back on and hold feed button till paper prints out.
  • Press and hold the feed button again for longer than a second. The printer should print paper
2. Change Settings
  • Find the option on the receipt paper saying Customize Value Settings
  • Note the number 3 highlighted in red below, if the number next to the Customize Value Settings is different follow along with specified number.
  • Press the feed button 3 times. Press the feed button one more time and hold.
  • More settings should feed out of the printer
